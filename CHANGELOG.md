## [v0.1.0] - 2026-01-25

### Highlights

- Initial release of structured-requirements for PRD, MRD, and TRD document management
- Integration with structureddocs and structured-goals for consistent rendering and goals alignment

### Added

- PRD (Product Requirements Document) types with personas, objectives, requirements, and roadmap
- MRD (Market Requirements Document) types with market analysis and competitive landscape
- TRD (Technical Requirements Document) types with architecture and technical specifications
- Marp slide renderer for PRD using structureddocs shared themes
- Goals alignment support for OKR and V2MOM frameworks via structured-goals
- Document completeness scoring and validation
- Persona library with common user archetypes
- Multiple view formats: executive summary, PM view, PR/FAQ, six-pager
- JSON Schema for PRD document validation
- CLI command for requirements document management
- Example PRD, MRD, and TRD documents

### Dependencies

- Add github.com/grokify/structureddocs v0.1.0 for shared Marp utilities
- Add github.com/grokify/structured-goals v0.1.0 for goals alignment

### Documentation

- MkDocs documentation site with getting started guides
- Document type references for PRD, MRD, and TRD
- Goals integration guides for OKR and V2MOM
- Feature documentation for scoring, completeness, and personas
